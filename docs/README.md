   # Healthcare Revenue Cycle Management - Data Engineering Platform
   
   ## Overview
   Production-grade data engineering platform for Healthcare RCM analytics using Azure cloud services, containerization, and Infrastructure as Code.
   
   ## Tech Stack
   - **IaC**: Terraform
   - **Container Orchestration**: Azure Kubernetes Service (AKS)
   - **Data Processing**: Azure Databricks
   - **Workflow Orchestration**: Apache Airflow
   - **Storage**: Azure Data Lake Storage Gen2
   - **Monitoring**: Prometheus + Grafana
